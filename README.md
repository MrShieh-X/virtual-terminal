# Commands
ls: List files </br>
exit: Exit </br>
echo: Echo a text </br>
cd: Change current directory </br>
su: Switch user </br>
co: Create an object </br>
cf: Create a file </br>
mkdir: Create a directory </br>
mkdirs: Create directories </br>
do: Delete an object </br>
df: Delete a file </br>
cp: Copy a file </br>
rn: Rename a file </br>
mv: Move a file </br>
